## インストール方法

1. 使いたい `.json` ファイルをダウンロード
2. ComfyUI を起動
3. 設定 → Appearance → Color Palette → 📂 アイコン（Import）
4. ダウンロードした JSON ファイルを選択
5. テーマ一覧から選んで適用
